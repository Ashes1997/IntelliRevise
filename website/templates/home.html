Welcome to IntelliRevise!

This is a website for generating flashcards from study notes.

Please head to the "Try it out" page to test out this function.

Key features of this website:

  • Built using Flask
  • Utilizes OpenAI's API
  • Can create, store, and authenticate users
  • Utilizes SQL-Alchemy to allow for the management of users, notes, and flashcards

Future plans:

  • Creation of a page to access previously created flashcards
  • Allowing for the generation of multiple-choice "exams" using already generated flashcards
